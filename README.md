💻 Programmer Quotes – Multi‑Language Edition

A curated collection of **quotes from famous programmers and tech personalities**, with tools to browse, search, add, and manage your own collection.  
Built in **7 programming languages** – perfect for daily inspiration, learning, or integrating into your projects.

## ✨ Features
- **Random quote** – get a random quote from the collection.
- **List all quotes** – view the entire collection with authors and categories.
- **Search by author** – filter quotes by a specific author.
- **Search by category** – filter quotes by topic (e.g., `debugging`, `wisdom`, `humor`).
- **Add quote** – add a new quote with author and category.
- **Remove quote** – delete a quote by its ID.
- **Save/Load** – persist quotes to a JSON file for sharing or backup.
- **Interactive CLI** – intuitive numbered menu.

## 🚀 How to Run
Each file is standalone – run it with the appropriate interpreter/compiler.

| Language | Command |
|----------|---------|
| Python   | `python quotes.py` |
| Go       | `go run quotes.go` |
| JavaScript | `node quotes.js` |
| C#       | `dotnet run` (or `csc Quotes.cs && Quotes.exe`) |
| Java     | `javac Quotes.java && java Quotes` |
| Ruby     | `ruby quotes.rb` |
| Swift    | `swift quotes.swift` |
